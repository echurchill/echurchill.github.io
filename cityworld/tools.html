Eclipse

GitHub

 

The source repository of choice, I like it so much I put my stuff there

 
FindBugs

 

This plugin keeps my code honest. The first time I used this it found so much stuff it took me a week to refactor everything to correct everything it found. I learned a lot from what it flagged and the explanations it gave. Great Stuff Indeed!

 
YourKit

 

While this is an odd name for a code profiler, it works great. In the past I found tools like this to be a pain to set up and get working with my code but this one worked first time out the door. Keep in mind that I am profiling a plugin (CityWorld) to a variant (Spigot) of a mod (Bukkit) to a server (MineCraft) while I played with a client (MineCraft). Crazy eh! They also support open source projects with a free license. So cool!

 

YourKit is kindly supporting open source projects with its full-featured Java Profiler. YourKit, LLC is the creator of innovative and intelligent tools for profiling Java and .NET applications. Take a look at YourKit's leading software products: YourKit Java Profiler and YourKit .NET Profiler.
